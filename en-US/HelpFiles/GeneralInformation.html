General Information
Vault QuickStart defines the workspace and storage location for CAD data, regardless of the used application: AutoCAD or Inventor.
This workspace is stored in the vault configuration and in the provided Inventor project file. To use the benefits of the automatic numbering, assigning to projects, classification and direct filling of properties logging on to the vault is always required.
 
 
A directory or project folder within the Vault workspace is maintained as default storage location during a user session until a new one is explicitly selected. For example, AutoCAD users get the first time they save their drawing the most recently used folder or project suggested – regardless of the last used application: AutoCAD or Inventor.
Note for users of Vault QuickStart 2014 and QuickStart 2015 RTM: Upgrade QS2014/2015 -> 2015 R1/R2
