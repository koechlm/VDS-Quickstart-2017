ERP Tab
Vault QuickStart enables the displaying of ERP data linked to documents or CAD components. This functionality allows you to display current ERP information associated with documents or construction files, without leaving the Vault explorer. An import of ERP data in the Vault Server is not necessary avoiding the risk of data redundancy.
Select a file or CAD component in the list view; if available, the current ERP information on the selected file will appear on the ERP tab.
Instructions for setting up the ERP tab can be found in the section Administration - ERP Tab.
