AutoCAD
Vault QuickStart for AutoCAD guides users - when saving a drawing - to the correct file storage, drawing numbering, and filling the title block. Templates are provided with matching border and title block.
 
Topics in this section

  • New File, first-time saving of Drawings
  • Edit properties in Datasheet / Title block

New Drawing
The first time you save a new drawing the AutoCAD dialog for the file category "AutoCAD drawing" will appear.
 Select the project / root folder and optionally a sector / sub-folder (s). For clarity, a limit of 3 levels is recommended.
The selection lists will show existing projects / folders and their subfolders. If you need a new directory, you can create this in Vault Explorer. The directory list will be automatically updated next time you select the folder in the drop-down list in the dialog.
 Use the My Shortcuts panel to save and restore a shortcut to the project- and subdirectories.
Activate / delete a shortcut with the right mouse button:

 Clear the selection of a subfolder by clicking on the X button.

Edit Datasheet / Title block
Drawing title block information is exclusively processed through the datasheet in Vault QuickStart environments. Use the Datasheet Command:
 
 
 The storage location is displayed read-only.
 The modification of properties is possible depending on the file status: checked out for editing.
 Renaming or moving the file is not provided in this context. Use the existing Rename or Move commands in Vault Explorer, since in this case all references will be checked and updated.

Legacy data
Existing drawings must be stored and indexed (recording of all Title Block properties) in Vault using the appropriate import methods, eg Vault Autoloader.
The first time you open the legacy drawing, make sure that the file is checked out immediately. At check out the title block information is included in the Quick Start sheet.
 
  
Any missing (mandatory) information can be added either in Vault Explorer (see Vault - Edit Datasheet) or in AutoCAD:
The first time you save the existing data, the data sheet appears when not all in the Vault Client configuration defined properties exist or are filled. The datasheet displays in edit mode. See also Edit Datasheet / Title Block.
