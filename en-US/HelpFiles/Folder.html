Folder
Vault QuickStart replaces the default command for creating folders by a command that creates the folder location and simultaneous add information to the folder or project.
 
 Topics in this section
  • Creating folders, projects
  • Numbering of projects
  • Editing folder / project datasheets
 
Create Folder / project...
Select project or folder via the category drop-down.
Creating folders: The Name Files, Title field and Description field are mandatory fields.
 
Creating Projects: Projects are numbered sequentially. The numbering applies throughout the vault, regardless of the directory structure. The project number is also the name of the project directory.

Numbering
The numbering scheme for projects can be freely defined. The dialog will always show the preview of the numbering scheme called "Project". If no scheme exists, specify a project number is offered as an option. The name of the project folder is taken from the Title field.
 
The activation and configuration of the numbering scheme "project" is reserved for administrators. See Object Numbering.

Edit Folder datasheet
You can edit all the properties in the project data sheet except the object-number.
Special Case: If you have existing directories, e.g. created by dragging and dropping from the Windows Explorer. When editing the data sheet, the folder name is taken as a suggestion in the title field. This can of course be overwritten.
For a large number of folders, we recommend the use the command “Edit Selected properties” because there you can fill Title and Description fields from the folder name for multiple folders in one single action:
