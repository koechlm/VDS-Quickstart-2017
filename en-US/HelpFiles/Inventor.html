New Components
  
The first time you save new parts and assemblies the Inventor dialog for the file category "3D components" will appear.
 
 Select the project / root folder and optionally a sector / sub-folder(s). For clarity, a limit of 3 levels is recommended.
The selection lists will show existing projects / folders and their subfolders. If you need a new directory, you can create this in Vault Explorer. The directory list will be automatically updated next time you select the folder in the drop-down list in the dialog.
 
 Use the My Shortcuts panel to save and restore a shortcut to the project- and subdirectories. Activate / delete a shortcut with the right mouse button:
 
The current selection of project and subdirectories can be saved as a new shortcut:
This shortcut will also be created in Vault explorer:
 
Note: In the CAD dialog only shortcuts within the CAD-workspace will be displayed.
 
 Clear the selection of a subfolder by clicking on the X button.
 Review the local storage location and file name.
Note: If automatic file naming is disabled (by the administrator), you will be prompted to enter the file name:

New Drawing / Presentation
  
The first time you save drawings or presentations an Inventor dialog box with corresponding property fields will appear.
Both dialog boxes (drawings and presentations) will respect the option “Generate file numbers for drawing and presentation files”:
  
 
Option - enabled: 
The selection for file number and storage location will be available. For a quick selection of the project assignment the shortcuts panel will be active. (See Shortcuts):
  
 Properties: These are - if available - taken from the model, but can be overridden for the drawing / presentation.
 Part, material or drawing number is assigned according to the chosen numbering scheme.
 
Option - disabled:
Location and file name taken are from the 3D model. These fields cannot be changed but are displayed for information. Consequently, there are no shortcuts to the selection of project-storage locations offered.
 
 Properties: These are - if available - taken from the model, but can be overridden for the drawing / presentation.
 Part, material or drawing number is taken from the linked model.
 
Special case: Drawings without a model view, for example with only sketches: Since it can’t get a drawing number and location from the model, the saving is only possible with the option "Create file numbers for drawing and presentation files" active:
 Properties: in this case, no properties can be read from the model and should be entered by the user.

Edit Datasheet
The datasheet of an open file (or active component in the context of assemblies) is accessed through the Datasheet command:
 
 The storage location is displayed read-only.
 The modification of properties is possible depending on the file status: checked out for editing.
Renaming or moving the file is not provided in this context. Use the existing Rename or Move commands in Vault Explorer, since in this case all references will be checked and updated.

Legacy data
The first time you save the existing data, the data sheet appears when not all in the Vault Client configuration defined properties exist or are filled. The datasheet displays in edit mode. See also Edit Datasheet.
