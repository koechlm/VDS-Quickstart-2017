Custom Object
Vault Quickstart for Vault Professional offers preconfigured Custom Objects (Collection of links) for the creating and editing of tasks, organisations, Persons / contacts and transmittals.

Create/Edit Tasks
Vault Quick Start uses the custom object "Task" to link documents and articles/BOMs with an activity. An task can be assigned via a link with one or more persons.
Create
Tasks are numbered sequentially, as long as the object numbering of this type is active:
If the numbering scheme is disabled by the administrator you can assign manually a number or text for the task naming.
Besides the Name field is the Title a required field.
 
Edit
In editing mode - and in case of active numbering scheme - you can not change the number (Name):
With an inactive object numbering the number can be changed manually. In this case, the title is equal to the object name.

Create/Edit Persons
Vault QuickStart uses the custom object "Persons" to link with other objects like Organisations or Tasks. Moreover, a person can also be linked to a document.
Create   
Persons are numbered sequentially, as long as the object numbering of this type is active. The first and last name are required fields.
 
If the numbering scheme is disabled by the administrator you can assign manually a number or text for the organisation naming. The first and last name are required fields.
 
Edit
In editing mode - and in case of active numbering scheme - you can not change the number (Name):
 
With an inactive object numbering the number can be changed manually.
